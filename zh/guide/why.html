# 为什么做了这款插件

cz-git 友好型 commitizen 的适配器，我的开发的心路历程在新窗口打开

  • cz-customizable在新窗口打开 :
    1. 需要额外添加配置文件。
    2. 仅支持上下选择是的交互方式。
    3. 可支持的习惯型配置项少。
  • cz-conventional-changelog在新窗口打开 :
    1. 支持的自定义配置项少。
    2. 交互方式不友好。
    3. 重复性输入的东西太多。
$ npm install -D cz-conventional-changelog
added 147 packages in 21s

$ npm install -D cz-git
added 1 package in 0.461s



 

  • git-cz在新窗口打开 :
    1. 可支持的自定义配置项少。
    2. 需要额外添加配置文件。
$ du -sh node_modules/*
148 MB	node_modules/git-cz
1.7 MB	node_modules/cz-git
